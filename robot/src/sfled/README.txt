Simple library for controll SmartFusion onboard leds for signaling and debugging.

Valid mss_gpio_ids are
MSS_GPIO_0
MSS_GPIO_1
MSS_GPIO_2
MSS_GPIO_3
MSS_GPIO_4
MSS_GPIO_5
MSS_GPIO_6
MSS_GPIO_7

Initialize the specified led with sfled_init

Initialize the specifie led with sfled_on

Toggle specified led with sfled_toggle

Turn off specified led with sfled_off

Flash specified led on/off specified number of times with specified delay between flashes in microseconds 
with sfled_flash
